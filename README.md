# portfolio
Portfolio pour présenter mes projets.

## Besoin
J'ai besoin d'un portfolio pour présenter mes projets.

## Analyse
Pour présenter mon portfolio je dois concevoir un site web avec comme fonctionnalités :
- la présentation de mes projets
- mes coordonnées pour me contacter

## Conception
### Structure
Le site sera composé de trois sections :
- le header avec une navigation et une brève phrase de présentation
- les projets présentés avec une image, une description et des liens
- une partie contact pour me retrouver

### Technologies
- HTML
- CSS
